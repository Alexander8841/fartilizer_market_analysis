# Анализ рынка удобрений (Карбамид)

Проект по прогнозированию цен на карбамид с использованием исторических ценовых данных и NLP-анализа отраслевых новостей.

## Цель проекта
Определить, улучшают ли рыночные новости прогнозирование цен на карбамид (urea_price) и какие типы новостей содержат наиболее полезную информацию.

## Структура проекта
- `data/` — исходные датасеты (цены и новости).
- `notebook.ipynb` — основной пайплайн исследования.
- `environment.yml` — зависимости окружения conda.
- `Business_Presentation.docx` — Бизнес-презентация решения.

## Установка и воспроизведение

### 1. Создание виртуального окружения
Убедитесь, что установлен Miniconda или Anaconda. 
Выполните команду:
```bash
conda env create -f environment.yml
```

### 2. Активация окружения
```bash
conda activate fertilizer_env
```

### 3. Запуск кода

Запустите Jupyter и откройте `notebook.ipynb`:
```bash
jupyter notebook
```
Выполните все ячейки последовательно ("Run All").

## Краткие результаты
В ходе исследования было проведено сравнение базовой модели (основанной только на числовых признаках) и гибридной модели, использующей текстовые признаки (NLP).
- **Лучшая модель:** CatBoost Regressor.
- **Метрики на отложенной выборке:** MAE = **38.14 $/т**, MAPE = **~7.6%**.
- **Бизнес-вывод:** Информационный фон (новости из открытых источников) практически не улучшает предсказательную способность алгоритма по сравнению с историческими ценами. Это связано с тем, что настоящие драйверы цен (результаты закрытых тендеров) публикуются в новостях постфактум. Базовая модель отлично улавливает инерцию рынка, но запаздывает при резких структурных шоках.

## Особенности данных и признаков
- **Целевая переменная:** Недельное изменение цены на карбамид.
- **Табличные признаки:** Лаги цен, скользящие средние (MA), волатильность, цены на сопутствующее сырье (аммиак).
- **Текстовые признаки:** Заголовки новостей очищены, лемматизированы (PyMorphy3) и векторизованы с помощью TF-IDF и эмбеддингов (SentenceTransformers `all-MiniLM-L6-v2`), после чего сжаты через PCA.

## Стек технологий
- **Обработка данных:** `pandas`, `numpy`
- **NLP (Обработка текста):** `nltk`, `pymorphy3`, `sentence-transformers`, `scikit-learn`
- **Моделирование:** `catboost`
- **Интерпретация моделей:** `shap`
- **Визуализация:** `matplotlib`, `seaborn`
