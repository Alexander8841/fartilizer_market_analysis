# Анализ рынка удобрений (Карбамид)

Проект по прогнозированию цен на карбамид с использованием исторических ценовых данных и NLP-анализа отраслевых новостей.

## Цель проекта
Определить, улучшают ли рыночные новости прогнозирование цен на карбамид (urea_price) и какие типы новостей содержат наиболее полезную информацию.

## Структура проекта
- `data/` — исходные датасеты (цены и новости).
- `notebook.ipynb` — основной пайплайн исследования.
- `requirements.txt` — зависимости.
- `Business_Presentation.docx` — Бизнес-презентация решения.

## Установка и воспроизведение

### 1. Клонирование репозитория
Склонируйте репозиторий на свой локальный компьютер и перейдите в папку проекта:
```bash
git clone https://github.com/Alexander8841/fartilizer_market_analysis.git
cd fartilizer_market_analysis
```

### 2. Создание и активация виртуального окружения
Рекомендуется использовать виртуальное окружение Python (venv):

**На Windows:**
```bash
python -m venv venv
venv\Scripts\activate
```

**На macOS/Linux:**
```bash
python3 -m venv venv
source venv/bin/activate
```

### 3. Установка зависимостей и регистрация ядра
Установите все необходимые библиотеки с помощью `pip`:
```bash
pip install -r requirements.txt
```

Зарегистрируйте виртуальное окружение в качестве ядра Jupyter (Jupyter Kernel), чтобы оно было доступно для выбора в ноутбуке:
```bash
python -m ipykernel install --user --name=fertilizer_env --display-name "Python (fertilizer_env)"
```

### 4. Запуск кода

Запустите Jupyter и откройте `notebook.ipynb`:
```bash
jupyter notebook
```
1. В верхнем меню Jupyter Notebook (или в правом верхнем углу в VS Code) выберите ядро **Python (fertilizer_env)** (`Kernel` -> `Change kernel` -> `Python (fertilizer_env)`).
2. Выполните все ячейки последовательно ("Run All").

## Краткие результаты
В ходе исследования было проведено сравнение базовой модели (основанной только на числовых признаках) и гибридной модели, использующей текстовые признаки (NLP).
- **Лучшая модель:** CatBoost Regressor (обученный на прогнозирование разностей цен).
- **Метрики на отложенной тестовой выборке:** MAE = **16.37 $/т**, MAPE = **4.21%** (успешно обходит наивный бенчмарк Naive 1-step с MAE = **17.81 $/т**).
- **Бизнес-вывод:** Информационный фон (новости из открытых источников) практически не улучшает предсказательную способность алгоритма по сравнению с историческими ценами. Это связано с тем, что важная информация публикуется в новостях постфактум. Базовая модель отлично улавливает динамику рынка, но запаздывает при резких скачках.

## Особенности данных и признаков
- **Целевая переменная:** Недельное изменение цены на карбамид (разностный таргет).
- **Табличные признаки:** Лаги цен, скользящие средние, волатильность, цены на сопутствующее сырье (аммиак).
- **Текстовые признаки:** Заголовки новостей очищены, лемматизированы (PyMorphy3) и векторизованы с помощью TF-IDF и эмбеддингов (SentenceTransformers `paraphrase-multilingual-MiniLM-L12-v2`), после чего сжаты через PCA.

## Стек технологий
- **Обработка данных:** `pandas`, `numpy`
- **NLP (Обработка текста):** `nltk`, `pymorphy3`, `sentence-transformers`, `scikit-learn`
- **Моделирование:** `catboost`
- **Интерпретация моделей:** `shap`
- **Визуализация:** `matplotlib`, `seaborn`
