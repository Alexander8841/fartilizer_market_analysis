# Описание данных (data dictionary)

Два файла. Данные синтетические, но приближены к реальному рынку удобрений. Частота —
недельная, период 2020 — середина 2026 (≈339 недель). Кодировка UTF-8.

> Данные «как в жизни»: возможны пропуски, шум и отдельные аномалии — корректная работа с
> ними входит в задание. **Важно:** прежде чем использовать колонку как признак, убедись, что
> её значение действительно доступно на дату строки (временная корректность, отсутствие утечки).

## Контекст рынка (если ты не из отрасли)
Цены на минеральные удобрения зависят от стоимости энергии и сырья, логистики и спроса со
стороны сельского хозяйства. В данных три семейства продуктов:
- **азотные** (карбамид, `urea`) — производятся из аммиака, а аммиак — из природного газа;
- **фосфорные** (`DAP`) — из фосфатного сырья и серы;
- **калийные** (`MOP`) — добыча калия; на цену сильно влияет баланс спроса и предложения.

Спрос на удобрения связан с ценами на агрокультуры (выше цены на урожай → фермеры активнее
покупают). Экспортные цены чувствительны к курсу валюты и стоимости доставки. Это общий
контекст; какие связи реально присутствуют в данных и как их использовать — выясняешь сам.

## prices.csv
Одна строка — одна неделя.

| Колонка | Что это |
|---|---|
| `date` | Дата (начало недели) |
| **`urea_price`** | **Целевая.** Цена карбамида (азотное удобрение), усл. $/т |
| **`dap_price`** | **Целевая.** Цена DAP (диаммонийфосфат, фосфорное), усл. $/т |
| **`mop_price`** | **Целевая.** Цена MOP (хлористый калий, калийное), усл. $/т |
| `gas_price` | Цена природного газа (индекс, TTF-подобный) |
| `ammonia_price` | Аммиак, усл. $/т (полупродукт для азотных удобрений) |
| `brent_oil` | Нефть Brent, $/баррель |
| `usd_rub` | Курс доллара к рублю |
| `freight_index` | Индекс морского фрахта (стоимость логистики) |
| `crop_price_index` | Индекс цен на агрокультуры (прокси спроса на удобрения) |
| `sulphur_price` | Сера, усл. $/т (сырьё для фосфатных удобрений) |
| `phosphate_rock_price` | Фосфатное сырьё, усл. $/т |
| `potash_supply_index` | Индекс предложения калия (ниже значение — туже рынок) |
| `consensus_price_urea` | Внешняя «консенсус-оценка» цены карбамида |

Целевые переменные — три цены (`urea_price`, `dap_price`, `mop_price`). Остальные колонки —
потенциальные факторы. Какие из них и как использовать — решаешь сам.

## market_news.csv
Поток коротких рыночных новостей.

| Колонка | Что это |
|---|---|
| `date` | Дата публикации |
| `source` | Издание/источник |
| `category` | Тема (энергоносители, калий, фосфаты, спрос/агрорынок, макро/валюта, логистика, прочее) |
| `headline` | Заголовок/краткая новость |

Часть новостей релевантна рынку, часть — информационный шум. Связь новостей с движением цен
(если она есть) — предмет твоего анализа.
